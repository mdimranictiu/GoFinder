##  Installation & Setup  
Follow these steps to run the project locally:  
 ## Client
### 1 Clone the Client Repository  
```
git clone https://github.com/mdimranictiu/GoFinder
cd go-finder-client
```

### 2 Install Dependencies
```
npm install
```
### 3 Start the Development Server
```
npm run dev
```
